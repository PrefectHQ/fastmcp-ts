Defined in: [client/auth.ts:571](https://github.com/PrefectHQ/fastmcp-ts/blob/e4b6038f642b529f601463a576229747afebe9a4/src/client/auth.ts#L571)

Configuration for [ClientCredentials](/api/client/classes/ClientCredentials). A discriminated union: supply
either a `clientSecret` (with an optional `authMethod`) or a `privateKey`
(with its `algorithm`) — the two are mutually exclusive and enforced at
compile time.
